# Structured Note Schema

Use this when the user asks for durable reading notes.

Recommended JSON fields:

```json
{
  "book": {
    "title": "",
    "author": "",
    "series": "",
    "publisher": "",
    "year": "",
    "source_file": "",
    "reading_scope": "",
    "reading_completed": ""
  },
  "methodology": {
    "scope_note": "",
    "source_base": [],
    "author_position": "",
    "limitations": ""
  },
  "key_arguments": [],
  "chapter_summaries": {},
  "critical_events": {},
  "people": {},
  "statistics": {},
  "quotations": [],
  "cross_references": [],
  "open_questions": [],
  "user_notes": ""
}
```

Keep notes evidence-oriented. Do not force every field if the source does not support it. For excerpts, keep quotations short and attach source location when available.
